# AI Resume Analyzer

A beginner-friendly Python project that analyzes how well a resume matches a job description using basic NLP techniques.

## Why this project?
Many candidates apply for jobs without knowing whether their resume aligns with the role. This project explores a simple, explainable approach to resume screening using text similarity.

## Features
- Resume vs job description comparison
- Keyword overlap analysis
- Match percentage using TF-IDF similarity

## Tech Stack
- Python
- Scikit-learn
- Basic NLP concepts

## How to run
```bash
pip install -r requirements.txt
python analyzer.py
